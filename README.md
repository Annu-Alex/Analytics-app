# Analytics Dashboard App 📊

A simple full-stack Analytics Dashboard application built using React, Node.js, Express, and API integration.

## 🚀 Features

- Full-stack web application (Frontend + Backend)
- React-based responsive dashboard UI
- Node.js and Express backend server
- REST API integration
- Fetches real-time data from backend API
- Sales data visualization using bar charts
- Displays product-wise sales information
- Shows analytics dashboard with clean UI
- Cross-Origin Resource Sharing (CORS) enabled
- Axios used for frontend-backend communication
- Fast development using Vite
- Simple and beginner-friendly project structure

## 🛠️ Technologies Used

### Frontend
- React.js

## ⚙️ Installation and Setup

### Clone Repository
Backend Setup
Go to backend folder:

cd backend

Install dependencies:

npm install

Run server:

node server.js

Backend runs on:

http://localhost:5000
Frontend Setup
Open another terminal:

cd frontend

Install dependencies:
npm install

Start application:
npm run dev

Frontend runs on:

http://localhost:5173
